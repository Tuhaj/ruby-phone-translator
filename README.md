## USAGE

Translates letters to numbers according to this schema:

other - 1

ABC   - 2

DEF   - 3

GHI   - 4

JKL   - 5

MNO   - 6

PQRS  - 7

TUV   - 8

WXYZ  - 9


```
$ ruby phone.rb 1-800-FLOWERS
1-800-3569377
$ ruby phone.rb 1-800-800
1-800-800
$ ruby phone.rb
Proper usage:
phone.rb number
```
